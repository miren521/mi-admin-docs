# 重新登录刷新权限

  • 侧边栏不会实时更新，需要重新登录拉取最新权限才会显示出来。

  • 如果没显示出来请检前面步骤查看是否已分配权限，菜单链接是否一致等。

  • 同时也可以刷新一下缓存
