接口文档调试

# Apidoc

  • 自动生成接口文档及在线调试功能使用的是 ThinkPHP-ApiDoc (opens new window)。
    请查看 ThinkPHP-ApiDoc文档 (opens new window)， 按要求编写注释即可自动生成接口文档。

  • 前端小伙伴，忽略上面一段话，然后可在后台 系统管理->接口文档 页面查看接口文档信息和在线调试接口。

  • 也可以点击右上角打开新标签页浏览文档
